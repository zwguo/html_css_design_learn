第1章 结构
目标：
	理解结构
	学习标记
	标签和元素
1.1、元素的定义
严格来说，一个元素包括起始标签、结束标签以及其中的内容。
1.2、特性让我们更了解元素
包括特性名称和特性值，比如<p lang="en-us"></p>，lang是特性名称，en-us是特性值。特性名称应该用小写字母。
1.3、html全称
HyperText Markup Language：超文本标记语言。
超文本是指允许你通过建立链接，是访问者简单地从一个页面跳转到另一个页面。
标记语言允许你对文本进行注释，而这些注释为文档的内容提供了更多的注解。
1.4、学习HTML就要学习哪些标签可用，它们有什么作用以及可以用在何处。
