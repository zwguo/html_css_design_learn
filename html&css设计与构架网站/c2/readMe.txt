目标：
	标题和段落
	粗体、斜体、强调
	结构化标记和语义化标记
2.1、结构化标记
可用来描述标题和段落的元素；
2.2、语义化标记
表达特定含义的标记。
2.3、标题
共有h1到h6等6个级别的标题。
2.4、段落
<p>的标签表示的就是段落。
2.5、粗体和斜体
<b>包裹的内容是粗体展示；
<i>包裹的内容是斜体展示；
<sup>包裹的是上标，<sub>包裹的是下标，比如水的化学式。
2.6、空白
当浏览器遇到两个或两个以上的连续空格时，只将其显示为一个空格。同样，当遇到一次换行时，浏览器也会将其看作一个空格。这一特性叫做白色空间折叠。
2.7、换行符和水平线
<br />表示换行符；
<hr />表示水平线。
2.8、加粗和强调
<strong>是加粗；
<em>是强调。
2.9、引用
<blockquote>是引用；
<abbr>是提示缩写词；
<cite>是引文来源；
<q>是引文，一般浏览器会用引号引起来；
<dfn>是下定义，一般斜体表示；
<address>包含页面设计者的详细信息，通常斜体，会换行；
<ins>用来显示已经插入到文档中的内容，而<del>元素则用来显示已经从文档中删除的文本，<s>是表示不准确但应该保留的信息。
