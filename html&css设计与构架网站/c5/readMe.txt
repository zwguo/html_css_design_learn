目标：
	如何向页面中添加图像
	选择正确的图像格式
	优化Web图像
5.1、添加图像
<img src="images/winter.jpg" alt="winter" title="this is a winter picture." />
src是地址，alt是当图片加载不出来时，出来的描述，title是鼠标悬停时的描述。
5.2、插入图像的位置
浏览器中有两种显示HTML元素的方式：
	块级元素总是另起一行显示；
	内联元素位于块级元素中，并且不会另起一行显示。
如果<img>元素后面是一个块级元素，则这个块级元素将在图像后的下一行开始。
如果<img>元素位于一个块级元素中，则文本及其他内联元素将环绕在图像的周围。
5.3、图像的位置调整
新的HTML5已经不支持<img>的align="left"这种设置。
其中的值包括：left、right、middle、top、bottom。
5.4、图像格式
JPEG：当图片包含不同的颜色时；
GIF、PNG：当图片中包含少量的颜色或者大面积的同色区域时；
当一个图片的某个区域填充了完全相同的颜色，它就被称为平面彩色。
当创建需要打印的图像时，最好以300点/英寸（DPI）或者更高的分辨率对齐进行保存，以保证清晰度。
5.5、矢量图像
矢量图和位图不同，通常由Adobe Illustrator软件创建。它可以任意的放大缩小而不失真。
目前网站的做法是把矢量图保存为它的位图版本，然后使用这个图像。
可伸缩矢量图形SVG是一个较新的格式，但目前还未普及。
5.6、GIF动画
可以显示几帧序列图像，因此可用来创建简单地动画。
5.7、HTML5图形和图形说明
图像往往带有说明，故HTML5引入了一个新的元素：<figure>，用<figcaption>进行说明.
