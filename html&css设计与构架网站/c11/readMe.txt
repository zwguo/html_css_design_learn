11.1、前景色
在CSS中有三种方法可以指定文本颜色：
	RGB值：rgb(100,100,90)
	十六进制编码：#ee3e80
	颜色名称：red,DarkCyan
注释使用/**/
11.2、背景色
CSS在处理每个HTML元素时都假设它们位于一个无形的盒子中，而background-color属性设置的正是这个盒子的背景色。
11.3、颜色解析
计算机屏幕上的每种颜色都由一定数量的红色、绿色、蓝色混合而成。
色调：色调很接近通俗意义上所说的颜色；
饱和度：饱和度是指颜色中灰色的含量。饱和度达到最大时，颜色中灰色的含量是零。饱和度最小时，颜色基本上就是灰色；
亮度：亮度是指颜色中黑色的含量。亮度达到最大时，黑色含量为零。亮度最小时，颜色就会非常俺。
11.4、对比度
为使文本清晰易读，在选取前景色和背景色时，一定要保证两者之间有足够的对比度。
查看对比度网站：https://snook.ca/technical/colour_contrast/colour.html#fg=687B33,bg=904233
11.5、透明度
CSS3中引入了opacity属性，允许你指定透明度，该值是一个介于0.0~1.0之间的数字。
CSS3中的rgba允许你像RGB一样指定颜色，不过最后的a是指透明度。
注意：opacity: 0.5;是设置这个元素及其内的内容。而rgba只是这个这个元素本身。
所以这两个值不同：
/*透明度*/
p.one{
	background-color: rgb(0,0,0);
	opacity: 0.5;
}
p.two{
	background-color: rgb(0,0,0);
	background-color: rgba(0,0,0,0.5);
}
11.6、HSL和HSLA
CSS3中引入了一种全新的颜色指定方式：HSL，比如：background-color: hsl(0,0%,78%);代表，色调0是一个0度到360度（圈）的一个角度，0%是饱和度；78%代表明度（明度和亮度不同，前者有白色和黑色，而后者只有黑色），0%代表黑色，50%表示校准色，100%代表白色；
而hsla则最后一位代表ALPHA：透明度，介于0~1.0之间。
